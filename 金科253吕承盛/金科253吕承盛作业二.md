1.  运行一行代码
    操作：将光标定位到要运行的代码行（无需选中整行），点击代码编辑区右上角的Run按钮，或使用快捷键：

运行多行代码
选中的多行代码：用鼠标选中连续/不连续的多行代码，点击Run按钮，或用快捷键：
Windows/Mac：Ctrl+Enter（Windows）/Cmd+Enter（Mac）（与单行运行快捷键相同，会自动识别选中内容）
整个脚本的所有代码：点击Run按钮旁的下拉箭头，选择Source（或直接用快捷键）：
Windows：Ctrl+Shift+S Mac：Cmd+Shift+S

8.注释方式

单行注释：用  #  开头， #  后的内容会被视为注释（不执行）；
多行注释：R语言本身没有原生的多行注释符号，通常通过每行加  #  实现，或用
 if(FALSE){}  包裹（不推荐，仅作临时注释）。

注释的快捷键

单行注释：选中要注释的行，按  Ctrl + Shift + C （Windows）/  Cmd +
Shift + C （Mac），会给选中行添加  # ；
取消单行注释：选中已注释的行，再次按同样的快捷键，会移除  # 。
